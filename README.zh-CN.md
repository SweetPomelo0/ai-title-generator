# AI 标题生成器 — 一键生成爆款标题的 Agent Skill

[English](README.md) | **中文**

> AI 标题 / 爆款标题生成 **skill**。把任意文章一键生成两档标题：
> ① **专业图文档**（公众号 / 知乎 / 人人都是产品经理）— 量子位 · 机器之心 · 新智元 三家风格
> ② **小红书口语档** — 第一人称 + emoji + 痛点钩子

## ✨ 它能做什么

- 一篇文章 → **公众号标题 / 知乎标题 / 小红书标题** 一次产出
- 量子位 / 机器之心 / 新智元 三家科技媒体风格，各主推 + 备选
- 自带 **平台 → 风格** 对应表，不用自己想该用哪档
- 内置标题策略：借爆款机制，但**钩子必真、不捏造**
- 适合中文内容创作者、自媒体、产品经理、AI 从业者

## 🚀 安装（`npx skills add`）

```bash
npx skills add SweetPomelo0/ai-title-generator
```

兼容 **Claude Code / Codex / Gemini CLI / GitHub Copilot** 等 skills 生态的 agent。

## 🎯 用法

把文章 / 选题丢给它，说 **"起标题"**。要单独某一档或某一家，说一声即可。

## 📦 示例

**输入：** 一篇"某 AI 模型更新、更强更便宜"的文章

**输出：**
```
【A 档 · 公众号 / 知乎 / 人人】
🔥 量子位    刚刚！这家甩出新模型：推理更强，价格还砍了
🎓 机器之心  新一代模型发布：推理能力提升，调用成本同步下降
⚡ 新智元    更强推理 + 更低价格：新模型一手登场，重新定义性价比

【B 档 · 小红书】
姐妹们这个 AI 又更新了😳 这次是真香：更聪明还更便宜
```

## 🗂 平台 → 风格

| 平台 | 用哪档 / 风格 |

| 小红书 | B 档 |
