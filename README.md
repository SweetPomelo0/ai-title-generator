# ai-title-allinone

一篇文章 / 一个选题 → **两档标题一次出齐**的 AI skill。

- **A 档 · 专业图文**（公众号 / 知乎 / 人人都是产品经理）：量子位 / 机器之心 / 新智元三家风格，各主推 + 备选
- **B 档 · 小红书口语档**：第一人称 + emoji + 痛点钩子
- 自带"平台 → 档位"对应表，不用自己想该用哪档
- 内置标题策略：机制可借、**钩子必真、不捏造**

## 安装

```bash
npx skills add SweetPomelo0/ai-title-allinone
```

## 用法

把文章 / 选题丢给它，说"起标题"。要单独某一档或某一家，说一声即可。
