# AI 标题工作台（两档全切）

## （可选）带入你的定位
在这里写一句你的身份 / 内容方向，标题会默认带上你的视角；不填就走通用。
> 例：「XX 领域一线从业者，内容偏真实实战 / 失败复盘。」

## 铁律（先记死，每次都过一遍）
1. **标题在"注意力市场"，正文在"信任市场"。** 标题尽管借爆款机制赢点击，不必为"保持克制人设"牺牲点击力——点击是流量的命根子。
2. **机制全借，钩子必真。** 好奇缺口 / 数字钩子 / 反差 / 损失厌恶 / 痛点反转随便用；但绝不编"刚刚 / 全球首个 / Nature / 具体金额 / 名人 / 榜单"——素材里没有就不写。用素材里的真实数字本身就够炸。
3. **若是信任型 IP / 专家号：标题开的支票，正文必须兑现。** 钓进来却没料 = 烧口碑。
4. **平台决定风格，不靠喜好。** 见下方对应表。
5. 保留真实英文名（GPT-5、Claude、DeepSeek…）；医疗/金融/法律/安全不夸张成确定性结论。

## Workflow
**第一步：解析成 fact card（两档共用一次）**
- 核心实体：模型 / 公司 / 产品 / 项目 / 人物 / 事件
- 动作：发布 / 搭建 / 复盘 / 翻车 / 突破 / 重估…
- 硬钩子：数字、速度、成本、对比、benchmark、金额、规模
- 权威背书：仅在素材真有时
- 利害：谁受益、谁被威胁、什么变了

**第二步：出两档**

### A 档 · 专业图文档（公众号 / 知乎 / 人人都是产品经理）
三家风格，各 1 主推 + 1 备选：
- 🔥 **量子位** = 情绪炸裂标题党：强时效开场 / 冒号分段 / 痛点反转 / 数字，可上感叹号、引号。（素材无名人则不蹭名人）
- 🎓 **机器之心** = 学术严谨：团队 + 方法 + 成果 / 设问 / 技术名词 / 硬背书，**去感叹号**。（发"人人都是产品经理"首选这档）
- ⚡ **新智元** = 高密度冲击：实体堆叠 + 数字钩子 + 强转折，24–42 字。

### B 档 · 小红书口语档
1 主推 + 1 备选：短、口语、第一人称、emoji、强痛点 / 强利益、好奇钩子。标题即封面文案，目标是"刷到就停"。

**第三步：自检** —— 每条过铁律 2、3；数字是否真；A 档机器之心是否去了感叹号；B 档是否够口语、够第一人称。

## 平台 → 档位对应表
| 平台 | 用哪档 | 备注 |
|---|---|---|
| 公众号 | A 档（三家都行） | 可放开，量子位/新智元也吃得开 |
| 人人都是产品经理 | A 档 · 机器之心为主 | 偏干货方法论、克制 |
| 知乎 | A 档 · 机器之心/设问为主 | 搜索向，标题埋关键词 |
| 小红书 | B 档 | 完全不通用，必须单独翻译 |
| 视频号 / B站 | A 档钩子 → 改成口播开场 | 先写后讲 |
